# automatic_lead_assignment
This is automatic lead assignement project


# Database Setup
1. Create a MySQL database (e.g. ->  `crm`)
2. Run the schema script:
   -> Execute `sql/schema.sql` using MySQL Workbench
3. Load sample data:
   -> Execute `sql/seed.sql`
4. Configure environment variables(.env)
5. Start the application
    -> npm install 
    -> npm start
